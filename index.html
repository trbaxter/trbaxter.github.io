Fractional Calculus Computation API (in progress)

Photo: RainforestShane/ ShutterStock

This API calculates the derivative or integral of an expression to arbitrary order. Intended for modeling real-world phenomena with increased precision beyond traditional integer-order calculus.

Currently functional for integer-order derivatives of polynomial expressions.
Click the picture above to learn more.






Future ideas for features include:
  • Frontend interface using React, Redux, and Bootstrap to improve UX
  • PostgreSQL database using to save evaluated points for 2D & 3D plotting
